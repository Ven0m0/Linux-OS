# Linux-OS

A collection of scripts and resources for managing and customizing Linux distributions.

<details>

```bash

```

```bash
curl -fsSL https://raw.githubusercontent.com/Ven0m0/Linux-OS/main/Cachyos/Clean.sh | bash
```

